# Projeto de Webcam + OpenCV

- A sua câmera  será ligada e o seu rosto será detectado, com um retângulo desenhado ao redor dele.
- Ao apertar a tecla "s", a sua foto será salva no seu computador.
- Ao apertar a tecla "ESC", a câmera é fechada.
